# cb1-exam-extractor
Extracting exam questions from CB1 papers. Basically, it was majorly cooked to have to find all the questions from an excel spreadsheet... So I whipped up a python script to sepearate questions and answers, but also name them.

- Plan is to also whip up a frontend to display the split files which will allow for ordering by week, etc.  
- I vibe-coded like the whole second-half of this repo. For example, `markscheme.py` and `check_gaps.py` are both vibecoded.
- That's sorta it. When the frontend is done, I'll link it.
- To try out the repo, just make sure you use venv and `pip3 install -r requirements.txt` &larr; (I'm not sure if that's the right terminal command this is just off the top of my head.)
